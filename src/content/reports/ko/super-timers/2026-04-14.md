## 스펙 감사 결과

아래 2개 파일을 현재 코드에 맞게 업데이트함.

**`specs/08-apple-watch.md`** — Watch 타이머 실행 화면(세션 모드) 전면 재작성  
- 진행 링: accent 색상, 일시정지 시 회색, 링 위 알람 마커(울리기 전 = accent, 울린 후 = 회색)  
- 링 내부: 남은 시간 + "일시정지" 뱃지(주황)  
- 링 아래: 완료 예정 시각(실행 중만), 타이머 이름, 다음 알람 행  
- 하단 버튼 쌍: 일시정지/재개 + 취소  
- 완료 오버레이(`remainingSeconds <= 0`): checkmark 아이콘 + "완료!" + 닫기 버튼

**`specs/04-alarm-management.md`** — 알람 타임라인 및 종료 알람 편집 UI 업데이트  
- 드래그 핸들이 트랙 아래(centerY+26)에 분리 배치 → 손가락이 트랙을 가리지 않음  
- 핸들↔커서 연결선, 트랙 하단 범위 레이블(`"N분 전"` / `"종료"`) 추가  
- 종료 알람 편집: 전용 헤더 버튼 폐지 → 커서를 종료 위치로 이동하면 나타나는 "수정" 버튼으로 통합
